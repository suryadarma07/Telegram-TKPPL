SignIn
User dapat melakukan login atau sign ke dalam aplikasi menggunakan akun telegram dan akan diminta kode verifikasi pada aplikasi mobile sebagai validasi user.

Logout
Selain melakukan login, user juga dapat melakukan logout terhadap akun tersebut melalui menu setting.

Add new friend
Ketika memasuki menu Contact, maka akan muncul kontak yang sudah user simpan dan dapat menambah kontak hanya dengan klik "new contact" kemudian isi seluruh data yang diperlukan.

Chat
User dapat melakukan chat dengan kontak yang sudah ditambahkan.

Multiple Language
User dapat mengganti bahasa yang digunakan pada menu setting sesuai dengan bahasa yang tersedia.

Username
User dapat menambah username sebagai samaran dari nomor telepon.

Voice Chat
User dapat mengirim voice chat atau pesan suara hanya dengan menekan tombol mic yang ada pada room chat.

Reply Message
User dapat menandai pesan yang dibalas dengan mudah yaitu dengan menekan tombol reply setelah melakukan select message

New Group
User dapat membuat grup chat yang berisikan lebih dari dua orang pada menu yang tersedia
